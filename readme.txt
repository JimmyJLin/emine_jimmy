NA Base project

Using Angular and Node.


1. Pre-requisites.
1.1. Make sure node is installed on your system.
1.2. Using npm, go to the server folder, where the package.json is, and type: npm install.

2. To Run.
2.1. Go to the server subfolder, in console.
2.2. Type "nodemon index.js".
2.3. The server should start up, listening on the default port 8082.

3. Test the locally running app.
3.1. In a web browser, go to: localhost:8082.
3.2. You should see the Home page, and some links at the top to start going places.
